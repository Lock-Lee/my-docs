# TODO List

## ✅ TODO List Application

> A task management system built with **Next.js**, **Tailwind CSS**, and **Shadcn UI**, integrated with **Hasura** and **ProgateSQL**.  
> It supports creating, editing, deleting, and marking tasks as complete — all within a modern, responsive, and intuitive user interface.

---

## 📌 Key Features

- 🔐 **User authentication** with login and registration support
- ✍️ **Register a new account** with email and password
- 🔑 **Login securely** with credential validation
- ✅ **Add new tasks** with title, description, and priority selection
- 📝 **Edit existing tasks** at any time
- 🗑️ **Delete tasks** you no longer need
- 📂 **Task grouping by status**: separates tasks into “TODO” and “DONE”
- ✅ **Mark tasks as completed** with a single click
- 📅 **Display current date** at the top of the screen
- 📱 **Responsive design** for all screen sizes (mobile & desktop)
- 🎨 **Modern and intuitive UI** for a smooth user experience
